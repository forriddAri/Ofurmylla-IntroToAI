# Ultimate tic tac toe

## How to run

The project requires Python 3 and the `pygame` library.

### Easiest (single command)

From this folder, run:

```bash
python run_game.py
```

What this does:
- Tries to import `pygame`.
- If `pygame` is missing, installs it using `pip` and `requirements.txt`.
- Then starts the game by running `main.py`.
